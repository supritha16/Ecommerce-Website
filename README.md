# Ecommerce-Website
Welcome to an e-commerce website using PHP and MySQL database. 

Introduction: Welcome to MyStore, an e-commerce website developed using PHP and MySQL database. MyStore offers a seamless shopping experience for users to browse, purchase, and manage their orders. This README file provides an overview of the website's features, installation guide, and usage instructions.

Installation Guide:
Requirements:
1.XAMPP (Cross-Platform Apache, MySQL, PHP, and Perl):
XAMPP is a free and open-source cross-platform web server solution stack package developed by Apache Friends, consisting mainly of the Apache HTTP Server, MariaDB database, and interpreters for scripts written in the PHP and Perl programming languages.
2.Web Browser:Any modern web browser such as Google Chrome, Mozilla Firefox, or Microsoft Edge.
<img width="510" alt="Screenshot 2024-03-17 003659" src="https://github.com/supritha16/Ecommerce-Website/assets/99244765/23f73fef-b4b9-4cde-baef-8d7942236e72">
3.Database Setup:Import the mystore.sql file into your MySQL database to create the necessary tables.
<img width="700" alt="Screenshot 2024-03-17 003943" src="https://github.com/supritha16/Ecommerce-Website/assets/99244765/0e5d8770-d79b-4fb4-8ffe-646274342ef3">
4.Run the Application:Start your web server.
Access the MyStore website through your web browser.
